# Photo placeholders

These SVGs are placeholders sized 4:5 (portrait). Swap them for real images using the same filenames (or update the references in `index.html`).

Used by:
- The side-peek layer (`#side-peek-layer` in `index.html`) — 6 images that drift in from the left/right edges as the cursor approaches.
- The `things i'm into` gallery section — 6 images in a 3-column grid.

To replace: drop a real `photo-NN.jpg` or `photo-NN.png` next to the SVG and update the `src` in `index.html`. Aim for portrait (4:5) crops for visual consistency.
