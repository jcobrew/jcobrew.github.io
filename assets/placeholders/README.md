# Photo placeholders

These SVGs are placeholders sized 4:5 (portrait). Swap them for real images using the same filenames (or update the references where they're used).

Currently used as:
- Chapter hero images and chapter-list thumbnails (referenced from `_chapters/*.md` front matter via the `hero:` field).
- Inline galleries inside chapter markdown (e.g. `_chapters/chapter-two.md`).
- Project card thumbnails on `projects.html`.
- Lab card thumbnails (referenced from `_experiments/*.md` front matter via the `thumb:` field).

To replace: drop a real `photo-NN.jpg` or `photo-NN.png` next to the SVG and update the matching `src` / front-matter reference. Aim for portrait (4:5) crops for visual consistency.
